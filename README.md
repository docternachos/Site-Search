# 🔎 Site Search

A sleek, modern website launcher built with **GitHub Pages + Firebase**.

Site Search lets users browse websites and games in a clean interface, open them in an embedded viewer, and manage everything through a secure admin panel.

---

## ✨ Features

- 🌌 Modern **purple + black theme**
- 🔍 Fast search system
- 🖼️ Custom images for every site
- 📝 Custom website names
- 🌐 Embedded website viewer
- 🖥️ Fullscreen support
- 📱 Mobile-friendly UI
- ⚡ Real-time syncing across devices
- 🔐 Passcode-protected admin panel
- 🚀 Hosted with GitHub Pages

---

## 📸 How It Works

### Homepage

Users can browse and search websites using the built-in search bar.

Each card includes:

- Website image
- Website name
- Launch button

Clicking a site opens it in the Site Search player.

---

### Website Player

The player page attempts to embed websites directly inside Site Search.

Features:

- **Fullscreen button**
- **Back button**
- **Smart embed system**

If a website blocks embedding, Site Search automatically shows:

> This site cannot be embedded.

and provides an **Open Site** button.

---

### Admin Panel

Admins can manage websites from:

```txt
/admin.html
```

Features:

✅ Add websites  
✅ Delete websites  
✅ Edit information (future update)  
✅ Access from any device

To add a website:

1. Enter website name
2. Paste image URL
3. Paste website URL
4. Click **Add Site**

Changes update live for everyone.

---

## 🔐 Admin Access

The admin panel is protected by a passcode system.

Authorized users can:

- Add new websites
- Remove websites
- Manage the library

## ⚠️ Important Note

Some websites **cannot be embedded** because they block iframes using browser security settings.

This is normal.

When this happens, Site Search automatically provides an **Open Site** button.
